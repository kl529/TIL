# FrontEnd (Vue)

## 페이징 말고, 더보기로 게시판 설정하기
  - 더보기 버튼 누르면, 마지막 id를 기준으로 해서 api 불러오기
  ```
  handleClickMore() {
      this.pending = true
      const query = this.$getFilterQuery('api')

      query.last_report_id = this.lastReportId
      
      this.fetchReportList(query, true).then(() => {
        this.pending = false
      })
    }
  ```
